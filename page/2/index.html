从尾到头打印链表

从尾到头打印链表？

核心思想：栈和递归。

public class VerseLink {

private ListNode head;

//添加链表节点
public void add(int data){
    ListNode node = new ListNode(data);
    if(this.head == null){
        this.head = node;
    }else {
        this.head.addNode(node);
    }
}

//从头到尾打印链表节点
public void print(){
    if(this.head == null){
        System.out.println("链表为空");
    }else {
        this.head.printNode();
        System.out.println();
    }
}

//从尾到头打印链表节点(常规方法) 时间复杂度(O(n)) 空间复杂度(O(1))
public void versePrint(){

    if(head == null){
        System.out.println("链表为空");
        return;
    }

    ListNode p = head.next;
    ListNode q = null;
    head.next = null;

    while (p != null){
        q = p.next;
        p.next = head;
        head = p;
        p = q;
    }
    print();
}

//栈实现 时间复杂度:O(n) 空间复杂度:O(n)
public void versePrint1(){
    Stack<Integer> stack = new Stack<Integer>();

    ListNode p = head;
    if(p == null){
        System.out.println("链表为空");
        return;
    }

    while (p != null){
        stack.push(p.data);
        p = p.next;
    }

    while (!stack.empty()){
        System.out.print(stack.pop()+"-->");
    }
}

//递归实现
public void versePrint2(ListNode p){
    if(p == null){
        return;
    }
    versePrint2(p.next);
    System.out.print(p.data+"-->");
}


public static void main(String[] args){
    VerseLink link = new VerseLink();
    link.add(1);
    link.add(2);
    link.add(3);
    link.versePrint2(link.head);
}
}

求链表中的倒数第k个结点？

public class Link {

private ListNode head;

class ListNode{

    private int data;

    ListNode next;

    public ListNode(int data) {
        this.data = data;
        this.next = null;
    }

    public void addNode(ListNode newNode){

        //相当于尾插法
        if(this.next == null){
            this.next = newNode;
        }else {
            this.next.addNode(newNode);
        }
    }

    public void printNode(){
        System.out.print(this.data + "-->");
        if(this.next != null){
            this.next.printNode();
        }
        }
    }

    //添加链表节点
    public void add(int data){
    ListNode node = new ListNode(data);
        if(this.head == null){
            this.head = node;
        }else {
            this.head.addNode(node);
        }
    }

    //打印链表节点
        public void print(){
    if(this.head == null){
        System.out.println("链表为空");
    }else {
        this.head.printNode();
        System.out.println();
    }
    }

    //链表中的倒数第 K 个结点
    public ListNode FindKthToTail(ListNode head,int k){
    int len = 0;
    int i = 0;
    ListNode counter = head;
    ListNode p = head;
    ListNode pre = head;

    if(k <= 0){
        System.out.println("请输入自然数k！");
        //-1代表非正常退出
        System.exit(-1);
    }

    if(head == null) {
        System.out.println("链表不能为空！");
        System.exit(-1);
    }

    while(counter != null){
        len++;
        counter = counter.next;
    }

    if(k > len){
        System.out.println("k值大于了链表长度！");
        System.exit(-1);
    }

    //先走k步
    while (i < k){
        pre = pre.next;
        i++;
    }

    while (pre != null){
        p = p.next;
        pre = pre.next;
    }

    return p;
    }

    public static void main(String[] args){
    Link link = new Link();
    link.add(1);
    link.add(2);
    link.add(3);
    link.print();

    ListNode node = link.FindKthToTail(link.head,4);
    node.next = null;
    node.printNode();
    }
    }

合并两个有序数组

发表于 2018-07-22 | 分类于 数组 | 阅读次数:
  • 两个有序数组A1和A2，把A2的所有数字插入到A1？

核心思想：从数组末尾开始比较，指针依次往前挪。

//时间复杂度(O(n)) 空间复杂度(O(n))
public static int[] merge(int[] a,int[] b){

    if(a==null || b==null){
        System.out.println("输入两个数组");
    }

    if(a.length == 0)
        return b;

    if(b.length == 0)
        return a;

    int len1 = a.length;
    int len2 = b.length;

    int[] c = new int[len1+len2];
    int i = len1 - 1;
    int j = len2 - 1;

    while(i >= 0 && j >= 0){
        if(a[i] > b[j]){
            c[i+j+1] = a[i];
            i--;
        }else{
            c[i+j+1] = b[j];
            j--;
        }
    }

    while (i >= 0){
        c[i+j+1] = a[i];
        i--;
    }

    while(j >= 0){
        c[i+j+1] = b[j];
        j--;
    }

    return c;
}


//时间复杂度(O(n)) 空间复杂度(O(1)) 直接在第一个数组上操作
public static void merge1(int[] a,int[] b){

    if(a==null || b==null){
        System.out.println("输入两个数组");
    }

    int len1 = a.length;
    int len2 = b.length;

    int i = len1 - len2 -1;
    int j = len2 - 1;

    while(i >= 0 && j >= 0){
        if(a[i] > b[j]){
            a[i+j+1] = a[i];
            i--;
        }else{
            a[i+j+1] = b[j];
            j--;
        }
    }

    while(j >= 0){
        a[i+j+1] = b[j];
        j--;
    }
}

替换字符串字符

发表于 2018-07-20 | 分类于 字符串 | 阅读次数:
  • 将字符串中的’ ‘替换成”20%”？

核心思想：

1.从最后一字符开始从前遍历，最后一个字符是’\0’。

2.将字符串变为数组时，要在字符串数组后面添加’\0’。

public class ReplaceChar {

//时间复杂度：O(n) 空间复杂度：O(n)
public void replace(char[] a,int len){

    //len = 1 时就一个字符'\0'
    if(a == null || len<2){
        System.out.println("输入长度大于0的字符串");
        System.exit(-1);
    }

    int oldLen = 0;
    int count = 0;
    int newLen;
    for(int i = 0;a[i] !='\0';i++){
        if(a[i] == ' ')
            count++;
        oldLen++;
    }

    newLen = oldLen + 2*count;

    if(newLen > len)
        return;

    int i = oldLen;//从'\0'开始
    int j = newLen;

    while (i >= 0 && i < j){

        if(a[i] == ' '){
            a[j--] = '0';
            a[j--] = '2';
            a[j--] = '%';
        }else {
            //注意 j 越界的情况
            a[j--] = a[i];
        }

        i--;
    }
}

public static void main(String[] args) {

    String s = "abcd";

    //多了个'\0'
    char oldChar[] = new char[s.length()+1];

    int counter = 0;
    for (int i = 0; i<s.length(); i++) {
        if (s.charAt(i) == ' ')
            counter++;
        oldChar[i] = s.charAt(i);
    }

    //字符串结束标志
    oldChar[s.length()] = '\0';

    //里面已经包含了'\0'结束符
    int totalLen = oldChar.length + counter*2;

    //数组扩容
    char[] newChar = new char[totalLen];
    for(int i = 0;i<oldChar.length;i++){
        newChar[i] = oldChar[i];
    }

    new ReplaceChar().replace(newChar,totalLen);

    for(int i = 0;newChar[i] != '\0';i++){
        System.out.print(newChar[i]);
    }

}
}

二维数组的查找

发表于 2018-07-19 | 分类于 数组 | 阅读次数:

二维数组，每一行从左到右递增，每一列从上到下递增，是否包含指定 key？

核心思想：

1.右上角的数字与要查找的数字比较。

2.右上角 > key 去列 否则 去行

public int findElement(int[][] arr,int n,int m,int search){

    int row = n;
    int col = m;
    int key = search;
    int i = 0;

    if(arr == null){
        System.out.println("请输入一个二维数组");
        return 0;
    }

    if(n <= 0 || m <=0){
        System.out.println("输入两个自然数");
    }

    while(i < row && col >0){
        //去列
        if(arr[i][col-1] > key){
            col--;
        }else if(arr[i][col-1] < key){
            i++;
        }else {
            //找到元素
            return arr[i][col-1];
        }

        //只剩一行
        if(i == row-1){
            for(int j=0;j<col;j++){
                if(arr[i][j] == key) {
                    return arr[i][j];
                }
            }

            return 0;
        }

        //只剩一列
        if(col == 1){
            for(int j = 0;j<row;j++){
                if(arr[j][0] == key)
                    return arr[j][0];
            }

            return 0;
        }
    }

    //不存在
    return 0;
}

数组中的重复数字

发表于 2018-07-18 | 分类于 数组 | 阅读次数:

长度为n的数组里面的所有数字都在0~n-1的范围内，找出数组中的重复数字？

核心思想：

1.数组下标 = 数组元素

2.目标交换：num[i] = m

swap: m <---> num[m] 直到 num[i] == i

public class OptDuplication {

//哈希表的方法解决，时间复杂度：O(n) 空间复杂度：O(n)
boolean findDup1(int[] nums,int len,int[] duplication){

    if(nums == null || len <= 0){
        System.out.println("数组必须存在且不能为空!");
        return  false;
    }

    for(int i = 0;i<len;i++){
        if(nums[i]<0 || nums[i]>len-1) {
            System.out.println("数组元素应在0 ~ len-1");
            return false;
        }
    }

    //哈希表
    int[] count = new int[len];
    for(int i = 0;i<len;i++){
        if(count[nums[i]] == 1){
            duplication[0] = nums[i];
            return true;
        }

        count[nums[i]]++;
    }

    return false;

}

boolean findDup(int[] nums,int len,int[] duplication){

    if(nums == null || len <= 0){
        System.out.println("数组必须存在且不能为空!");
        return  false;
    }

    for(int i = 0;i<len;i++){
        if(nums[i]<0 || nums[i]>len-1) {
            System.out.println("数组元素应在0 ~ len-1");
            return false;
        }
    }

    int temp;
    for(int i = 0;i<len;i++){
        while (i != nums[i]){
            //nums[i]=m 说明 m 所在的下标是 i 和 m 所以重复了
            if(nums[i] == nums[nums[i]]){
                duplication[0]=nums[i];
                return true;
            }else {
                //交换num[i] 和 num[num[i]]
                temp = nums[i];
                nums[i] = nums[nums[i]];
                nums[temp] = temp;
            }
        }
    }

    System.out.println("没有重复数字");
    return false;
}

长度为n+1的数组里面的所有数字都在1~n的范围内，所以必定存在重复的数字，找出数组中的重复数字？

核心思想：

1.二分法的运用，加入了统计数字个数的方法

2.循环结束条件：start == end

public class DuplicationVersion {

//时间复杂度:O(nlogn) 空间复杂度:O(1)
boolean findDup(int[] nums,int len,int[] duplication){

    if(nums == null || len <= 0){
        System.out.println("数组必须存在且不能为空!");
        return  false;
    }

    for(int i = 0;i<len;i++){
        if(nums[i]<=0 || nums[i]>len-1) {
            System.out.println("数组元素应在1 ~ "+ (len-1)+ "之间");
            return false;
        }
    }

    int start = 1;
    int end = len-1;
    int middle;
    while(start <=  end){

        middle = (start + end)/2;
        int number = calNum(nums,len,start,middle);

        //结束条件
        if(start == end){
            if(number > 1){
                duplication[0] = start;
                return true;
            }else{
                break;
            }
        }

        if(middle - start + 1 < number){
            end = middle;
        }else {
            start = middle + 1;
        }
    }

    System.out.println("没有重复数字");
    return false;
}

private int calNum(int[] nums, int len, int start, int end) {
    int count = 0;
    for(int i = 0;i<len;i++){
        if(nums[i] >= start && nums[i] <= end){
            count++;
        }
    }
    return count;
}
 }

单例模式

发表于 2018-07-17 | 分类于 Java设计模式 | 阅读次数:

  • 单例模式指？

    只生成一个对象，避免重复创建对象。

  • 单例模式中构造函数为什么私有化？

    1.构造函数私有化后，那么只能通过这个类的内部函数才能构造这个类的对象。

    2.将生成类对象的方法用static修饰的原因是：因为静态方法不需要依赖实例化的对象。

  • 单例模式中volatile变量的作用？

    当一个共享变量被volatile修饰时，它会保证修改的值立即更新到主存，其他线程去内存中读取新值。

  • 单例模式中为什么要使用双重检查索(DCL)？

    因为在多线程的环境下，所有线程在调用getSingleton()方法时都必须在synchronized这里排队，而加锁是一个非常耗时的操作，这导致单例模式的效率非常低。我们只需要在实例还没有创建之前添加加锁操作。

  • 静态内部类方法创建单例模式的好处？

    按需创建实例（饿汉式模式：在类加载时就完成了初始化），提高了内存的利用效率。

    这种方式的特点是：Singleton类被装载了，instance不一定被初始化，只有显示通过调用getInstance方法时，才会显示装载SingletonHolder类，从而实例化instance。

  • 栈内存和堆内存的比较？

**堆内存存的内容：**数组元素、类对象、类中的非静态成员变量（实例变量）、包装类数据(Integer等)、

**栈内存内容：**

1.方法中声明的变量(因为方法执行时，被分配的内存就在栈中,即方法栈)：分基本数据类型变量和引用变量，前者的变量名和值都存在方法栈中，后者声明的变量存放在栈中，所指的对象放在堆中。

2.对象的引用

**方法区：**线程共享的内存区域。类里面的静态成员和静态方法、

**常量池：**方法区的一块内存。

**静态域：**方法区的一块内存。存放类中的静态成员变量

  • 局部变量保存在栈内存中，随方法的消失而消失;成员变量存储在堆中的对象里，由垃圾回收器负责回收。

  • 包装类数据Integer、String等等存在于堆中。
