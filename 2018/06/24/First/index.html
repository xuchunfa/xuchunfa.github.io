Hexo + Github部署网站关键点

发表于 2018-06-24 | | 阅读次数:

第一步

安装好Git和node.js,然后配置系统环境变量，我用nvm安装node.js时自动配好了环境变量，记得配Git的环境变量

请看这里

第二步

新建文件夹存放博客内容和相关配置文件，例如在D盘建立Blog文件夹，进到文件夹，右键Git Bash,安装hexo
npm install -g hexo-cli

第三步

hexo的初始化
hexo init 默认是在当下文件夹，即在D:\Blog\下

第四步

根据package.json文件中的dependencies的配置文件安装依赖
npm install

第五步

生成部署文件，启动本地服务
hexo generate/g 
hexo server/s 在 http://localhost:4000/查看

第六步

在Github上建库，库名为username.github.io,其中username是你github账号的用户名，

第七步

  • 生成ssh密钥

    `ssh -keygen -t rsa -C "username@**.com"`

  • 拷贝密钥
    在Git Bash 中查看并拷贝cat ~/.ssh/id_rsa.pub

  • 添加密钥到Github
    [https://github.com/settings/keys](https://github.com/settings/keys)下add new ssh key

  • 测试连接
    ssh git@github.com

第八步

修改D:\Blog下的_config.yml配置文件
deploy:
    type: git
    repo: ssh://git@github.com/username/username.github.io
    branch: master
 注意冒号后的空格

第九步

安装Git包
npm install hexo-deployer-git --save
如果你是第一次部署的话可能会出现fatal: Not a git repository
此时初始化 git init

第十步

部署 hexo deploy/d
Hello World
